Со всех трех подключений я получаю данные о котировках которые генерирует генератор. Котировки отдаются в многопоточном режиме.

Требование постоянного PING выполнено. Если какой-то клиент перестает слать PING команды то сервер перестает слать данные по UDP о котировках.
